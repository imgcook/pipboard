# Pipboard

![build](https://github.com/imgcook/pipboard/workflows/build/badge.svg?branch=main)

[pipboard](https://ml.imgcook.com) 是一个线上交互式教程站点，通过线上交互式的体验，可视化的将模型的整个训练过程表达给使用者，使体验者可以快速了解模型相关的知识。

## 使用

### 一，收集数据

可通过摄像头或上传文件进行样本收集
